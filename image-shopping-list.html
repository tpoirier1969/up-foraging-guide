Image Shopping List

Upper Michigan Foraging Guide visual asset checklist. This is meant to live in the repo so you can reach it from anywhere and know exactly what images still need to be hunted down.

Project: Upper Michigan Foraging Guide Doc version: v1.3 Purpose: image coverage, not just image collecting

Master Rule

For each entry, aim for 1 thumbnail focused on the edible or medicinal part, plus 2 to 4 support images, plus 1 habitat/context shot if you can get it. The goal is useful visual coverage, not a bloated shoebox of random photos.

Best target per entry

  • 1 thumbnail, edible part first
  • 2 to 4 support images
  • 1 habitat or whole-organism shot
  • Total target: 4 to 6 images

Priority order

  • Phase 1: part, leaf/stem/bark, context
  • Phase 2: ripe vs unripe, processing, extra angles
  • Phase 3: lookalikes, seasonal changes

Current Species Shopping List

American Persimmon fruit entry

Already have: ripe fruit on branch.

  • Need: unripe fruit. This is critical because ripeness changes usefulness and taste dramatically.
  • Leaf close-up, top and underside if possible
  • Bark close-up
  • Whole tree or tree form in context
  • Nice-to-have: split fruit interior
  • Nice-to-have: fallen fruit on ground

Oak / Acorns nut entry

Already have: acorns.

  • Need: leaf close-up. Oak group identification falls apart fast without it.
  • Bark close-up
  • Whole tree silhouette or branch form
  • Acorn variation across species if you can get it
  • Need: processing visual, such as leaching setup or stages, because acorns are not a pluck-and-chew situation.

Barberry shrub entry

Already have: berries.

  • Need: thorns close-up. That is one of the main field clues and also a hazard clue.
  • Leaf close-up
  • Whole shrub shot
  • Berry cluster plus thorns in the same frame if possible
  • Nice-to-have: fall color variation

Wild Bergamot / Monarda flower-herb entry

Already have: flower in habitat.

  • Need: leaf close-up
  • Need: stem shot showing square stem if possible
  • Flower close-up from above
  • Patch or colony view
  • Optional: dried flower head or later-season view

Birch tree host and sap entry

Already have: bark close-up.

  • Need: leaf close-up
  • Whole tree
  • Need: sap tapping setup or sap collection context if this is being treated as a usable-part entry
  • Grove or multiple trunks shot
  • Optional: winter bark context

Beech / Beech Nuts tree and nut entry

Already have: nuts plus husk.

  • Need: bark close-up. Smooth gray bark is one of the easiest beech clues.
  • Leaf close-up
  • Whole tree or branch form
  • Opened husk with nuts clearly visible

Mushroom Coverage Rules

Mushrooms are the divas of the app. A pretty top view alone is not enough. Every serious mushroom entry needs the clues people actually use in the woods.

  • Cap top view
  • Underside view, meaning gills, pores, teeth, or smooth surface
  • Stem view
  • Growth context, such as ground, log, stump, living tree, or branch
  • Cross-section if possible
  • Young versus mature stage if possible
  • Staining or bruising reaction if the species does that

Host tree coverage

If it grows on wood, try to get the mushroom and the host surface in the same photo. Bonus points if the host tree is identifiable. The app needs broad host choices like hardwood and conifer, but the deeper dream is exact host trees when the evidence exists.

Mistake prevention shots

For tricky mushrooms, get the image that prevents a dumb mistake. That might be underside texture, ring, staining, or a toxic lookalike comparison. The glamour shot is not the hero there.

Naming Rule

Use a filename pattern that says what the image is without making you open it:

  • persimmon_fruit_ripe.jpg
  • barberry_thorn_detail.jpg
  • wild-bergamot_stem_square.jpg
  • birch_bark_close.jpg
