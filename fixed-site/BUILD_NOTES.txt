Starter rare/endangered build files for the Upper Michigan Foraging Guide.

Included:
- index-masterlist-v20.html
- js/constants-mainfix.js
- js/state.js
- styles-rare-v1.css
- data/rare-species-v1.json
- foraging_rare_supabase_setup.sql

This starter package includes the new section shell, project-safe Supabase names,
and seed data for rare plants plus a non-legal fungi watchlist.

Not included yet:
- full app wiring files
- map JS module
- sightings save/load UI code
