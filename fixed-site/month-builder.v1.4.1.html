WNMU month builder v1.4.15 duration/start-pair parser

Drop in one or both weekly-grid PDFs, parse them locally in your browser, then publish the imported month directly to the app-specific Supabase tables. This version reads ProTrack duration/start-time pairs such as 00:28:46 09:30:00, uses the first value as duration, and uses the second value to confirm the row time. Notes, tags, and rectangle notes are not modified by this import tool.

What this builder changes

  • Writes imported month schedule and verification JSON directly to Supabase.
  • Marks the imported month current only for the channel you publish.
  • Blocks partial-month parses and hidden-hole guesses; visible holes must remain visible.
  • Does not touch existing notes, tags, rectangle notes, archives, or backups.
